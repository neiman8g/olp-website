## How Buyers Search
Our buyers are engineers and specifiers searching for very specific products. They search by:
- UL listing number + product type + capacity
- Application + equipment type
- Manufacturer location (for Buy American compliance)

## Metadata Pattern
Every page should follow this structure:

```
Title: [Product/Category] | [Key Spec] | Online Power
Description: [What it is] + [key spec] + [certification] + [differentiator]. [Location/credibility].
```

### Examples
```
Title: Emergency Lighting Inverters | UL924 | 35W–400kW | Online Power
Desc: Centralized emergency lighting inverters from 35W to 400kW. UL924 listed, pure sine wave output, <4ms transfer time. Built in Los Angeles. ISO 9001.

Title: Elevator Backup Systems | UL924 | 24–240kW | Online Power
Desc: Three-phase elevator backup power for hydraulic, traction, and regenerative elevators. UL 924 and UL 1778 dual-listed. Zero transfer time. 2-hour runtime.
```

## Rules
- Always include the UL listing number in the title
- Always include power range or key spec
- Always mention "manufacturer" or "built in Los Angeles" — signals we're the OEM, not a reseller
- Never use generic phrases like "power protection solutions" or "industry-leading"
- Product page titles should include the model name and series
